# Lab: Learn git commands

## Objective

In this lab, we will learn how to perform the following activities:

- Create the repository
- Connect to the repository
- Create a branch
- Change the branch
- Stage the changes
- Commit the changes
- Pull the changes
- Create a pull request
- Peer review the code
- Merge the branches
- Create the issues
